WHAT ARE WEB SOCKETS
- Communication between a client (browser) and a server
- Bidirectional (Data flows both ways)
- Allows data flow in real time


USE OF WEB SOCKETS
- Multiplayer browser games
- Collaborative code/document editing
- Live text for sports/news websites
- Online drawing canvas
- Real time applications with multiple users

TERMINAL COMMANDS:
"node -v" to see if NodeJS is already installed and find the version
"node index" to start the server
"" to restart the server
